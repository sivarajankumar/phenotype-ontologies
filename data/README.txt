
## Files

 * SPECIES-gene-to-phenotype-FILTERCODE.txt 
    - 2 column file mapping gene ID to phenotype ID
    - OBO-style IDs are used
    - MOD IDs are used for genes
 * SPECIES-genotype-to-phenotype-FILTERCODE.txt 
    - 2 column file mapping genotype ID to phenotype ID
    - OBO-style IDs are used
 * SPECIES-gene-labels-FILTERCODE.txt 

## Filter codes

 * NOFILTER
 * BF - Basic filter. Single affected gene genotypes, only w.t. environments.

##All files are ABNORMAL phenotypes only; all "normal" phenotypes are removed.
